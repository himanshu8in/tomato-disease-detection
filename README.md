This project uses **EfficientNet CNN** to detect tomato leaf diseases using images.

- 10 tomato disease classes
- EfficientNetB0 architecture
- Streamlit web application
- Real-time prediction with confidence score

## Dataset
PlantVillage Tomato Dataset

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit
